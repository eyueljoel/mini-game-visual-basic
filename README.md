# 🎮 Rock Paper Scissors Game

A simple and interactive **Rock Paper Scissors** game developed using **Visual Basic** in **Visual Studio**.

This project was created to practice GUI application development, game logic implementation, and event-driven programming using Visual Basic.

---

## 🚀 Features

* Player vs Computer gameplay
* Random computer choice generation
* Score tracking system
* Interactive graphical interface
* Simple and user-friendly design

---

## 🛠️ Technologies Used

* Visual Basic (.NET)
* Visual Studio
* Windows Forms

---

## ▶️ How to Run

1. Open the project in **Visual Studio**
2. Build the solution
3. Run the application

---

## 🎯 Game Rules

* Rock beats Scissors
* Scissors beats Paper
* Paper beats Rock

The computer randomly selects a move and the winner is determined automatically.
